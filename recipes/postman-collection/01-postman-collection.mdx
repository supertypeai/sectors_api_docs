Welcome to the **Sectors Financial API** documentation! This guide is designed to help developers and integrators get started with our API, which provides market and analytical data specific to Indonesia. Whether you're building financial applications, conducting market research, or integrating data into your platform, this API will provide you with the tools you need.

In this tutorial, we'll walk you through how to use our **Postman Collection** to explore and test our API endpoints.

---

## Introduction

The **Sectors Financial API** offers a wide range of financial and analytical data for the Indonesian market. You can access real-time stock prices, historical data, market trends, and more. To make it easier for developers to interact with our API, we have created a **Postman Collection** that contains all the necessary endpoints and example requests.

---

## Prerequisites

Before you begin, ensure you have the following:

- **Postman**: Download and install [Postman](https://www.postman.com/downloads/) if you haven't already.
- **API Key**: You will need an API key to authenticate your requests. If you don't have one, please go to [Sectors Financial API](https://sectors.app/api) to generate API key for your account.
- **Basic Knowledge of APIs**: Familiarity with RESTful APIs and JSON formats is recommended.

---

## Getting Started with Postman

Postman is a powerful tool for testing and interacting with APIs. It allows you to send HTTP requests, view responses, and organize your API calls into collections.

If you're new to Postman, check out their official [documentation](https://learning.postman.com/docs/getting-started/introduction/) to get familiar with the interface.

---

## Importing the Sectors Financial API Postman Collection

To start using our API, follow these steps to import the **Sectors Financial API Postman Collection**:

1. **Download the Postman Collection**:
   - Click here [Latest Postman Collection](https://github.com/supertypeai/sectors_api_docs/blob/main/recipes/postman-collection/json/Sectors_API.postman_collection.json) to download the latest all-in-one version.

2. **Open Postman**:
   - Launch the Postman application on your computer.

3. **Import the Collection**:
   - Click on the **Import** button in the top-left corner of the Postman interface.
   - Select the downloaded `Sectors_API.postman_collection.json` file and click **Open**.
   - The collection will now appear in your Postman workspace under the **Collections** tab.

## API Endpoints

Our API provides several endpoints to fetch different types of financial data. Here is the full [API Reference](../../api-references) for you to check the details

## Collection Structure

The collection is named **Sectors API** and contains a **v2** folder. Under the **v2** folder, endpoints are grouped by resource type:

- **v2**
  - [**broker-activity**](../../api-references/v2/indonesia/brokers/broker-activity-by-code): Endpoints for tracking individual broker activity and top accumulations/distributions.
  - [**broker-summary**](../../api-references/v2/indonesia/brokers/broker-summary-by-symbol): Endpoints for broker summaries per stock symbol.
  - [**brokers**](../../api-references/v2/indonesia/brokers/broker-registry): General broker registry and daily rankings.
  - [**companies**](../../api-references/v2/indonesia/screener/companies): Structured screener, list companies with segments, and top changes.
  - [**company**](../../api-references/v2/indonesia/report/company-report): Individual company details including corporate actions, segments, quarterly dates, shareholder composition, and company reports.
  - [**daily**](../../api-references/v2/indonesia/transaction/daily): Daily pricing and trading data.
  - [**filings**](../../api-references/v2/indonesia/news/filings): Public company financial filings.
  - [**financials**](../../api-references/v2/indonesia/report/quarterly-financials): Historical and quarterly financial statements.
  - [**foreign-flow**](../../api-references/v2/indonesia/brokers/foreign-flow-by-symbol): Foreign investor transaction flows.
  - [**free-float**](../../api-references/v2/indonesia/screener/free-float): Free float data.
  - [**idx-total**](../../api-references/v2/indonesia/transaction/idx-total): Aggregated Indonesia stock exchange stats.
  - [**index-daily**](../../api-references/v2/indonesia/transaction/index-daily): Index performance daily data.
  - [**industries**](../../api-references/v2/indonesia/helper-list/industries): List of IDX industries.
  - [**klse**](../../api-references/v2/malaysia/klse-companies): Malaysia market (KLSE) companies, reports, and sectors.
  - [**listing-performance**](../../api-references/v2/indonesia/ipo/listing-performance): Performance of IPOs/listings.
  - [**mining**](../../api-references/v2/mining/companies/mining-companies): Mining intelligence service (commodities, company performance, ownership, contracts, exports, licenses, mine sites, total production).
  - [**most-traded**](../../api-references/v2/indonesia/ranking/most-traded): Most traded stocks statistics.
  - [**news**](../../api-references/v2/indonesia/news/news): Financial and market news.
  - [**sgx**](../../api-references/v2/singapore/screener/sgx-companies): Singapore market (SGX) buybacks, companies, reports, daily performance, filings, news, and sectors.
  - [**subindustries**](../../api-references/v2/indonesia/helper-list/subindustries): List of IDX sub-industries.
  - [**subsector**](../../api-references/v2/indonesia/report/sector-report): Report by subsector.
  - [**subsectors**](../../api-references/v2/indonesia/helper-list/subsectors): List of IDX subsectors.
  - [**suspensions**](../../api-references/v2/indonesia/news/suspensions): Trading suspensions.
  - [**tags**](../../api-references/v2/indonesia/helper-list/tags): Analytics tags and indicators.

Each request is pre-configured with the necessary parameters and variables.

## Usage & Authorization

To authenticate your requests, you need to apply the API key authorization at the collection level (the parent folder):

1. **Get your API key**: Go to [sectors.app/api](https://sectors.app/api) to generate/copy your API key.
2. **Open the main folder**: In Postman, open/click the main collection folder (**Sectors API**).
3. **Go to the Authorization tab**: Select the **Authorization** tab.
4. **Configure the variable**: Hover over the `{{apiKey}}` placeholder in the authorization value field, click to set it as a collection variable, and paste the API key you retrieved from sectors.app.
5. **Alternative**: You can also switch to the **Variables** tab of the **Sectors API** folder, set the value of the `apiKey` variable directly, and save.
