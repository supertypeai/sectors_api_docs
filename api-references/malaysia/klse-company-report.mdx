### Header

<ParamField header="Authorization" type="string" required>
  Authorization header that should be filled with your Sectors Financial API
  key.
</ParamField>

### Path

<ParamField path="ticker" type="string" required>
  Ticker of the company for which the report is to be retrieved.
  <AccordionGroup>
    <Accordion title="Get the available ticker">
      Get the available `ticker` from the [KLSE Companies by
      Sector](./klse-companies-by-sector) endpoint.
    </Accordion>
    <Accordion title="Examples of the ticker">
      `1155`, `1023`, `1295`, `5347`, `5225`
    </Accordion>
    <Accordion title="Format">
      A valid `ticker` consists of 4 numbers.
    </Accordion>
  </AccordionGroup>
</ParamField>

### Response

<ResponseField name="symbol" type="string">
  The ticker of the company.
</ResponseField>

<ResponseField name="name" type="string">
  The full name of the company.
</ResponseField>

<ResponseField name="overview" type="object">
  An overview of the key metrics of the given ticker.
</ResponseField>

<ResponseField name="valuation" type="object">
  Valuation metrics and related information of the given ticker.
</ResponseField>

<ResponseField name="financials" type="object">
  Historical financials and key financial ratios of the given ticker.
</ResponseField>

<ResponseField name="dividend" type="object">
  Dividend information of the given ticker.
</ResponseField>
