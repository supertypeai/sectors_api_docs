---
title: "Mining Company Financials"
api: "GET /v2/mining/companies/financials/{slug}/"
description: "Retrieves financial records (assets, revenue, profit) for a mining company by its slug for a single year (defaults to latest)."
---

<ResponseExample>
```json 200
{
    "year": 2024,
    "available_years": [2022, 2023, 2024],
    "data": {
        "idx_ticker": "ITMG.JK",
        "name": "PT Indo Tambang Megah",
        "slug": "indo-tambang-megah",
        "year": 2024,
        "assets_usd": 2406.54,
        "revenue_usd": 2304.5,
        "revenue_breakdown": {
            "Coal": 2288.76,
            "Services": 5.29
        },
        "cost_of_revenue_usd": 1605.69,
        "cost_of_revenue_breakdown": {
            "Royalty": 260.19
        },
        "net_profit_usd": 375.6
    }
}
```

```json 404
{
  "error": "Company not found."
}
```

</ResponseExample>

### Header

<ParamField header="Authorization" type="string" required>
  Authorization header that should be filled with your Sectors Financial API
  key.
</ParamField>

### Path Parameters

<ParamField path="slug" type="string" required>
  The unique identifier (slug) of the mining company.
</ParamField>

### Query Parameters

<ParamField query="year" type="integer">
  The year to retrieve (e.g., 2024). Defaults to the latest available year.
</ParamField>

### Response

<ResponseField name="year" type="integer">
  The year for which financial data is being returned.
</ResponseField>

<ResponseField name="available_years" type="integer[]">
  A list of all years where financial data is available for this company.
</ResponseField>

<ResponseField name="data" type="object">
  <Expandable title="Financial Data properties">
    <ResponseField name="idx_ticker" type="string | null">
      The company's IDX ticker.
    </ResponseField>
    <ResponseField name="name" type="string">
      The name of the company.
    </ResponseField>
    <ResponseField name="slug" type="string">
      The slug of the company.
    </ResponseField>
    <ResponseField name="year" type="integer">
      The year of the financial record.
    </ResponseField>
    <ResponseField name="assets_usd" type="number | null">
      Total assets for the year (in USD).
    </ResponseField>
    <ResponseField name="revenue_usd" type="number | null">
      Total revenue for the year (in USD).
    </ResponseField>
    <ResponseField name="revenue_breakdown" type="object">
      JSON object detailing the sources of revenue.
    </ResponseField>
    <ResponseField name="cost_of_revenue_usd" type="number | null">
      Total cost of revenue for the year (in USD).
    </ResponseField>
    <ResponseField name="cost_of_revenue_breakdown" type="object">
      JSON object detailing the components of the cost of revenue.
    </ResponseField>
    <ResponseField name="net_profit_usd" type="number | null">
      Net profit for the year (in USD).
    </ResponseField>
  </Expandable>
</ResponseField>
