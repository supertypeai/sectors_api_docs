---
title: "Top Export Destinations"
api: "GET /v2/mining/exports/"
description: "Ranks countries by total export value for a given year and commodity, showing the top destinations for exports."
---

<ResponseExample>
```json 200
[
    {
        "country": "Singapore",
        "export_usd": 2319208305,
        "export_volume_bps": 1500000.5,
        "export_volume_esdm": 1450000.2,
        "volume_unit": "Mt"
    },
    {
        "country": "Switzerland",
        "export_usd": 404012143,
        "export_volume_bps": 50000.0,
        "export_volume_esdm": 48000.5,
        "volume_unit": "Mt"
    }
]
```

```json 400
{
  "error": "Query parameters 'year' and 'commodity_type' are required."
}
```

</ResponseExample>

### Header

<ParamField header="Authorization" type="string" required>
  Authorization header that should be filled with your Sectors Financial API
  key.
</ParamField>

### Query Parameters

<ParamField query="year" type="integer" required>
  The year to analyze export data for (e.g., `2024`).
</ParamField>
<ParamField query="commodity_type" type="string" required>
  The commodity to analyze (e.g., `Gold`, `Nickel`, `Coal`).
</ParamField>
<ParamField query="limit" type="integer" default="20">
  The number of top countries to return. Maximum value is 30.
</ParamField>

### Response

<ResponseField name="[]" type="ExportDestination">
  <Expandable title="Export Destination properties">
    <ResponseField name="country" type="string">
      The destination country.
    </ResponseField>
    <ResponseField name="export_usd" type="number">
      The total value of exports in USD for the specified commodity and year to
      that country.
    </ResponseField>
    <ResponseField name="export_volume_bps" type="number">
      The volume of exports reported by BPS.
    </ResponseField>
    <ResponseField name="export_volume_esdm" type="number">
      The volume of exports reported by ESDM.
    </ResponseField>
    <ResponseField name="volume_unit" type="string">
      The unit of measurement for volume (e.g., "Mt").
    </ResponseField>
  </Expandable>
</ResponseField>
