---
title: "List Global Commodity Data"
api: "GET /v2/mining/global-commodity/"
description: "Retrieves global commodity data, including production, reserves, and trade information. Allows filtering by country and commodity type."
---

<ResponseExample>
```json 200
[
    {
        "country": "Australia",
        "commodity_type": "Gold",
        "resources_reserves": 55000000,
        "resources_reserves_share": 18.5,
        "resources_reserves_unit": "ton",
        "export_import_usd": {
            "2023": {
                "export": 20.9,
                "import": 1.2
            }
        },
        "production_volume": {
            "2023": 296.1,
            "2024": 284.0

        "production_share": {
            "2023": 8.13,
```

```json 400
{
  "error": "UNSUPPORTED_PARAMETERS",
  "message": "Unsupported parameters: offset."

```

</ResponseExample>

### Header

<ParamField header="Authorization" type="string" required>
  Authorization header that should be filled with your Sectors Financial API
  key.
</ParamField>

### Query Parameters

<ParamField query="country" type="string">
  Filter by a specific country (exact match, e.g., `Australia`).
</ParamField>
<ParamField query="commodity_type" type="string">
  Filter by a specific commodity (e.g., `Gold`, `Nickel`, `Coal`).
</ParamField>
<ParamField query="limit" type="integer" default="20">
  Number of results to return. Default: 20, Max: 30.
</ParamField>

The response is a list of commodity data records.

<ResponseField name="country" type="string">
  The name of the country.
</ResponseField>
<ResponseField name="commodity_type" type="string">
  The type of commodity.
</ResponseField>
<ResponseField name="resources_reserves" type="number | null">
  The absolute volume of resources and reserves.
</ResponseField>
<ResponseField name="resources_reserves_share" type="number | null">
  The country's percentage share of global resources and reserves.
</ResponseField>
<ResponseField name="resources_reserves_unit" type="string">
  Unit for resources and reserves.
</ResponseField>
<ResponseField name="export_import_usd" type="object | null">
  JSON object keyed by year: `{"2023": {"export": ..., "import": ...}}`.
</ResponseField>
<ResponseField name="production_volume" type="object | null">
  A dictionary map of years (`YYYY`) to production volumes.
</ResponseField>
<ResponseField name="production_share" type="object | null">
  A dictionary map of years (`YYYY`) to global production percentage shares.
</ResponseField>
<ResponseField name="production_volume_unit" type="string">
  Unit for production volume.
</ResponseField>
