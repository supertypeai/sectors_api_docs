## 2026-01-09

- Adjusted all mining related endpoints to use `/mining` instead of `/insider`
- Updated Postman collection and recipe to match the latest API version
- Multiple refinements to the Companies Screener V2 documentation, including:
  - Adding \_q capability for quarterly data.
  - Clarifying eps columns and sort orders.
  - Updated company report documentation for SGX and KLSE

## 2025-10-24

- Added new endpoint to retrieve global commodity data, including production, reserves, and trade information

  - [https://api.sectors.app/v2/global-commodity-data/](./global-commodity)

- Added new endpoint to retrieve mining license auctions

  - [https://api.sectors.app/v2/mining-license-auctions/](./mining-license-auctions)

- Added new endpoint to retrieve and aggregate sales destination data for a specific company ticker

  - [https://api.sectors.app/v2/sales-destination/](./sales-destination)

- Updated insider-related endpoints to use the `/v2/insider/` base path for improved consistency

## 2025-09-25

- Official launching of the second version of Sectors API.
