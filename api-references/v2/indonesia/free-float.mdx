---
title: "Free Float Market Analysis"
api: "GET /v2/free-float/"
description: "Retrieve free float percentage for companies, with optional filtering by sector, sub-sector, industry, or sub-industry."
---

The Free Float API provides the percentage of a company's shares that are held by the public and available for trading on the secondary market.

<Note>
  Free float is calculated as the `share_percentage` of the 'Public' entry in a
  company's major shareholders list.
</Note>

### Query Parameters

<ParamField query="sector" type="string">
  The slug of the sector for which the report is to be
  retrieved.
  <AccordionGroup>
    <Accordion title="Get the available sector">
      Get the available `sector` from the
      [Subsectors](./helper-list/subsectors) endpoint.
    </Accordion>
    <Accordion title="Examples of the sector">
      `infrastructures`, `transportation-logistic`, `healthcare`
    </Accordion>
    <Accordion title="Format">
      The API endpoint only accepts `sector` in the **kebab case** format (lowercase
      and separated by "-").
    </Accordion>
  </AccordionGroup>
</ParamField>

<ParamField query="sub_sector" type="string">
  The slug of the subsector for which the report is to be
  retrieved.
  <AccordionGroup>
    <Accordion title="Get the available subsector">
      Get the available `sub_sector` from the
      [Subsectors](./helper-list/subsectors) endpoint.
    </Accordion>
    <Accordion title="Examples of the subsector">
      `banks`, `basic-materials`, `food-beverage`
    </Accordion>
    <Accordion title="Format">
      The API endpoint only accepts `sub_sector` in the **kebab case** format (lowercase
      and separated by "-").
    </Accordion>
  </AccordionGroup>
</ParamField>

<ParamField query="industry" type="string">
  The slug of the industry for which the report is to be
  retrieved.
  <AccordionGroup>
    <Accordion title="Get the available industry">
      Get the available `industry` from the [Industries](./helper-list/industries)
      endpoint.
    </Accordion>
    <Accordion title="Examples of the industry">
      `oil-gas`, `electrical`, `chemicals`
    </Accordion>
    <Accordion title="Format">
      The API endpoint only accepts `industry` in the **kebab case** format
      (lowercase and separated by "-").
    </Accordion>
  </AccordionGroup>
</ParamField> 

<ParamField query="sub_industry" type="string">
  The slug of the subindustry for which the report is to be
  retrieved.
  <AccordionGroup>
    <Accordion title="Get the available sub_industry">
      Get the available `sub_industry` from the [Subindustries](./helper-list/subindustries)
      endpoint.
    </Accordion>
    <Accordion title="Examples of the sub_industry">
      `paper`, `life-insurance`, `personal-care-products`, `software`, `gold`
    </Accordion>
    <Accordion title="Format">
      The API endpoint only accepts `sub_industry` in the **kebab case** format
      (lowercase and separated by "-").
    </Accordion>
  </AccordionGroup>
</ParamField> 

<Warning>
  Query parameters are **mutually exclusive**. You should provide at most one
  filter parameter in your request.
</Warning>

### Response

The response is a flat list of companies, ordered by `free_float` in descending order.

<ResponseField name="symbol" type="string">
  The unique stock ticker symbol (e.g., `BBCA.JK`).
</ResponseField>

<ResponseField name="company_name" type="string">
  The full legal name of the company.
</ResponseField>

<ResponseField name="free_float" type="number">
  The free float percentage expressed as a decimal (e.g., `0.4512` represents
  45.12%).
</ResponseField>

<ResponseExample>

```json 200
[
  {
    "symbol": "BBCA.JK",
    "company_name": "Bank Central Asia Tbk",
    "free_float": 0.4512
  },
  {
    "symbol": "BBRI.JK",
    "company_name": "Bank Rakyat Indonesia (Persero) Tbk",
    "free_float": 0.4321
  }
]
```

</ResponseExample>
