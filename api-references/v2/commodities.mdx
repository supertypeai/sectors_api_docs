---
title: "List Commodities"
api: "GET /v2/mining/commodities/"
description: "Lists all commodities available in the price database, including their data coverage (monthly data points and date range)."
---

<ResponseExample>
```json 200
[
    {
        "name": "Gold",
        "data_points": 702,
        "earliest_date": "1968-01-01",
        "latest_date": "2026-06-30"
    },
    {
        "name": "Coal (HBA 1)",
        "data_points": 48,
        "earliest_date": "2023-01-01",
        "latest_date": "2026-05-31"
    }
]
```
</ResponseExample>

### Header

<ParamField header="Authorization" type="string" required>
  Authorization header that should be filled with your Sectors Financial API
  key.
</ParamField>

### Response

<ResponseField name="[]" type="Commodity">
  <Expandable title="Commodity properties">
    <ResponseField name="name" type="string">
      The name of the commodity (e.g., "Gold", "Nickel", "Coal (HBA 1)").
    </ResponseField>
    <ResponseField name="data_points" type="integer">
      Total number of monthly/bi-weekly data points available in the price
      database.
    </ResponseField>
    <ResponseField name="earliest_date" type="string">
      The earliest date for which price data is available (YYYY-MM-DD).
    </ResponseField>
    <ResponseField name="latest_date" type="string">
      The latest date for which price data is available (YYYY-MM-DD).
    </ResponseField>
  </Expandable>
</ResponseField>
