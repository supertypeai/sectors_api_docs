`GET /v2/mining/sites/`

<ResponseExample>

            "slug": "batubara-benhes-3"

</ResponseExample>

### Header

<ParamField header="Authorization" type="string" required>
  Authorization header that should be filled with your Sectors Financial API
  key.
</ParamField>

### Query Parameters

<ParamField query="company" type="string">
  Filter by company slug.
</ParamField>
<ParamField query="province" type="string">
  Filter mining sites by exact province name (e.g., Kalimantan Timur).
</ParamField>
<ParamField query="commodity_type" type="string">
  Filter sites by commodity (e.g., `Coal`, `Nickel`, `Gold`, `Copper`).
  Case-insensitive.
</ParamField>
<ParamField query="year" type="integer">
  Filter by reporting year.
</ParamField>
<ParamField query="min_production" type="number">
  Filter for sites with a production volume greater than or equal to this value.
</ParamField>
<ParamField query="order_by" type="string" default="-year">
  Sort field. Allowed: `production_volume`, `year`, `strip_ratio`. Prefix with
  `-` for descending.
</ParamField>
<ParamField query="limit" type="integer" default="20">
  Number of results to return. Maximum value is 30.
</ParamField>
<ParamField query="offset" type="integer" default="0">
  Number of results to skip. Used for pagination.
</ParamField>

## Mining Site Detail

`GET /v2/mining/sites/{slug}/`

Returns full details for a single mining site by its slug, including parsed resources/reserves and location (with lat/long).

<ResponseExample>
```json 200
{
    "name": "Batubara Benhes 3",
    "project_name": "Project Alpha",
    "year": 2024,
    "commodity_type": "Coal",
    "production_volume": 1200.5,
    "unit": "Mt",
    "overburden_removal_volume": 5000.0,
    "strip_ratio": 4.17,
    "resources_reserves": [
        {
            "category": "Measured Resources",
            "volume": 50.5,
            "unit": "Mt"
        }
    ],
    "location": {
        "province": "Kalimantan Timur",
        "city": "Kutai Timur",
        "latitude": 1.234,
        "longitude": 116.567
    },
    "company_slug": "adaro-energy",
    "company_name": "PT Adaro Energy Indonesia Tbk",
    "slug": "batubara-benhes-3"
}
```
</ResponseExample>

### Path Parameters

<ParamField path="slug" type="string" required>
  The unique identifier for the mining site.
</ParamField>
