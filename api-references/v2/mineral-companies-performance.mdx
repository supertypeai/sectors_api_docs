---
title: "Mineral Company Performance"
api: "GET /v2/insider/mineral-companies/{company_id}/performance/"
description: "Retrieves and compares performance data for a specific mineral company across multiple years."
---

<ResponseExample>
```json 200
[
    {
        "year": 2022,
        "commodity_type": "Coal",
        "commodity_sub_type": "Sub-bituminous & Metallurgical Coal",
        "commodity_stats": "{\"unit\": \"Mt\", \"production_volume\": 49.1, ...}"
    },
    {
        "year": 2019,
        "commodity_type": "Coal",
        "commodity_sub_type": "Sub-bituminous & Metallurgical Coal",
        "commodity_stats": "{\"unit\": \"Mt\", \"production_volume\": 51.6, ...}"
    }
]
```

```json 400
{
  "error": "Invalid years parameter. Please provide a comma-separated list of integers."
}
```

```json 429
{
  "code": 429,
  "message": "Rate limit exceeded."
}
```

</ResponseExample>

### Header

<ParamField header="Authorization" type="string" required>
  Authorization header that should be filled with your Sectors Financial API
  key.
</ParamField>

### Path Parameters

<ParamField path="company_id" type="integer" required>
  The unique identifier of the mineral company.
</ParamField>

### Query Parameters

<ParamField query="years" type="string">
  A comma-separated list of years to retrieve performance data for (e.g.,
  `2022,2019`). If omitted, data for all available years will be returned.
</ParamField>

### Response

<ResponseField name="[]" type="PerformanceRecord">
  <Expandable title="Performance Record properties">
    <ResponseField name="year" type="integer">
      The year the performance data was recorded for.
    </ResponseField>
    <ResponseField name="commodity_type" type="string">
      The general type of commodity.
    </ResponseField>
    <ResponseField name="commodity_sub_type" type="string">
      The specific sub-type of the commodity.
    </ResponseField>
    <ResponseField name="commodity_stats" type="string (JSON Object)">
      A JSON string containing detailed statistics.
      <Expandable title="Commodity Stats Structure">
        <ResponseField name="unit" type="string">
          Unit of measurement (e.g., "Mt").
        </ResponseField>
        <ResponseField name="production_volume" type="number">
          Volume of production.
        </ResponseField>
        <ResponseField name="sales_volume" type="number">
          Volume of sales.
        </ResponseField>
        <ResponseField name="overburden_removal_volume" type="number">
          Volume of overburden removed.
        </ResponseField>
        <ResponseField name="strip_ratio" type="number">
          Calculated strip ratio.
        </ResponseField>
        <ResponseField name="resources_reserves" type="object">
          Object containing resource and reserve data.
        </ResponseField>
        <ResponseField name="product" type="array">
          Array of product specifications.
        </ResponseField>
      </Expandable>
    </ResponseField>
  </Expandable>
</ResponseField>
