### Header

<ParamField header="Authorization" type="string" required>
  Authorization header that should be filled with your Sectors Financial API
  key.
</ParamField>

### Query Parameters

<ParamField query="commodity_type" type="string">
  Filter by commodity (e.g., `Nickel`, `Coal`). Case-insensitive.
</ParamField>
<ParamField query="province" type="string">
  Filter by province (e.g., `Kalimantan Tengah`). Case-insensitive.
</ParamField>
<ParamField query="area_type" type="string">
  Filter by area type (e.g., `WIUPK`). Case-insensitive.
</ParamField>
<ParamField query="status" type="string">
  Filter by auction status (e.g., `Lelang Selesai`). Case-insensitive.
</ParamField>
<ParamField query="participant" type="string">
  Filter auctions where a company name (partial match) participated.
</ParamField>
<ParamField query="qualified" type="boolean">
  When `true`, only return auctions where the participant passed qualification.
  Requires `participant`.
</ParamField>
<ParamField query="min_participants" type="integer">
  Only return auctions with at least this many participants.
</ParamField>
<ParamField query="order_by" type="string" default="-winner_date">
  Sort field. Allowed: `winner_date`, `participant_count`, `licensed_area_ha`,
  `commodity_type`. Prefix with `-` for descending.
</ParamField>
<ParamField query="limit" type="integer" default="20">
  Number of results to return. Maximum value is 30.
</ParamField>
<ParamField query="offset" type="integer" default="0">
  Number of results to skip. Used for pagination.
</ParamField>

## Auction Detail

`GET /v2/mining/license-auctions/{wiup_code}/`

Retrieves the full record for a single mining license auction, including the parsed phases timeline and participant qualification list.

<ResponseExample>
```json 200
{
    "wiup_code": "3411003442014022",
    "commodity_type": "Coal",
    "province": "Kalimantan Tengah",
    "city": "Kotawaringin Timur",
    "company_name": "PT Natai Baru Raya",
    "winner_date": "2024-05-30",
    "auction_status": "Lelang Selesai",
    "participant_count": 6,
    "winner": true,
    "phases": [
        {
            "phase": "Pengumuman Lelang",
            "date": "2024-01-01",
            "status": "Selesai"
        }
    ],
    "participants": [
        {
            "name": "PT Natai Baru Raya",
            "qualified": "LOLOS"
        }
    ]
}
```
</ResponseExample>

### Path Parameters

<ParamField path="wiup_code" type="string" required>
  The unique identifier for the auction.
</ParamField>
