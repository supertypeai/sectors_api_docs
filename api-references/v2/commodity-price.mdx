---
title: "Commodity Price History"
api: "GET /v2/mining/commodities/{name}/price/"
description: "Retrieves historical price data for a commodity by year range. Maximum range is 3 years."
---

<ResponseExample>
```json 200
[
    {
        "name": "Coal",
        "date": "2022-01-01",
        "price_usd_per_ton": 158.5
    },
    {
        "name": "Coal",
        "date": "2022-02-01",
        "price_usd_per_ton": 188.38
    }
]
```

```json 400
{
  "error": "The date range must not exceed 365 days."
}
```

```json 404
{
  "error": "Commodity 'Gold' not found."
}
```

</ResponseExample>

### Header

<ParamField header="Authorization" type="string" required>
  Authorization header that should be filled with your Sectors Financial API
  key.
</ParamField>

### Path Parameters

<ParamField path="name" type="string" required>
  The name of the commodity to retrieve price data for (e.g., `Coal (HBA 1)`).
</ParamField>

### Query Parameters

<ParamField query="start_year" type="integer">
  Start year (e.g. 2022). Defaults to current year - 2.
</ParamField>
<ParamField query="end_year" type="integer">
  End year inclusive (e.g. 2024). Defaults to current year.
</ParamField>

### Response

The response is a list of price records sorted by date in ascending order.

<ResponseField name="name" type="string">
  The name of the commodity.
</ResponseField>

<ResponseField name="date" type="string">
  The date of the price entry (YYYY-MM-DD).
</ResponseField>

<ResponseField name="price_usd_per_ton" type="number">
  The price of the commodity in USD per ton.
</ResponseField>
