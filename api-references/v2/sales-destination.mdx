---
title: "Get Sales Destination Data"
api: "GET /v2/mineral/sales-destination/"
description: "Retrieves and aggregates sales destination data for a specific company ticker. The data is structured by year, then by country, providing a comprehensive overview of a company's sales distribution over time and across different locations."
---

### Header

<ParamField header="Authorization" type="string" required>
  Authorization header that should be filled with your Sectors Financial API
  key.
</ParamField>

### Query Parameters

<ParamField query="ticker" type="string" required>
  The company's stock ticker (e.g., 'ITMG.JK').
</ParamField>

### Response

<ResponseField name="[year]" type="object">
  An object where each key is a year, containing sales data for that period.
  <Expandable title="Year properties">
    <ResponseField name="[country]" type="object">
      An object where each key is a country name, detailing sales performance.
      <Expandable title="Country properties">
        <ResponseField name="revenue" type="number | null">
          The total revenue from sales to this country.
        </ResponseField>
        <ResponseField name="percentage_of_total_revenue" type="number | null">
          The percentage this country's revenue contributes to the total.
        </ResponseField>
        <ResponseField name="volume" type="number | null">
          The total sales volume to this country.
        </ResponseField>
        <ResponseField name="percentage_of_sales_volume" type="number | null">
          The percentage this country's volume contributes to the total.
        </ResponseField>
      </Expandable>
    </ResponseField>
  </Expandable>
</ResponseField>
