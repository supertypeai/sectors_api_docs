---
title: "National Resources and Reserves"
api: "GET /v2/resources-reserves/"
description: "Aggregates and lists national resources and reserves data, with options to filter by location, year, and commodity."
---

<ResponseExample>

```json 200
[
    {

        "year": 2024,

    {

    }

```

### Header

<ParamField header="Authorization" type="string" required>
  Authorization header that should be filled with your Sectors Financial API
  key.
</ParamField>

### Query Parameters

<ParamField query="province" type="string">
  Filter data by a specific province.
</ParamField>
<ParamField query="year" type="integer">
  Filter data by a specific year.
</ParamField>
<ParamField query="commodity_type" type="string">
  Filter data by a specific commodity (e.g., `Gold`).
</ParamField>
<ParamField query="limit" type="integer" default="20">
  Number of results to return. Maximum value is 30.
</ParamField>
