### Header

<ParamField header="Authorization" type="string" required>
  Authorization header that should be filled with your Sectors Financial API
  key.
</ParamField>

### Query Parameters

<ParamField query="keyword" type="string">
  A search query that matches against the company name, IDX ticker, slug, or key
  operations.
</ParamField>
<ParamField query="commodity_type" type="string">
  Filter companies by a specific commodity (e.g., `Coal`, `Nickel`, `Gold`).
  Case-insensitive.
</ParamField>
<ParamField query="company_type" type="string">
  Filter by company type. Valid values include: `Mine Owner`, `Contractor`,
  `Holding`, `Manufacturer`, `Trader`, `Consultant`.
</ParamField>
<ParamField query="has_financials" type="boolean">
  Set to `true` to only return companies with financial data available.
</ParamField>
<ParamField query="limit" type="integer" default="20">
  Number of results to return. Maximum value is 30.
</ParamField>
<ParamField query="offset" type="integer" default="0">
  Number of results to skip. Used for pagination.
</ParamField>
