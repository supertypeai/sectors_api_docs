## Overview

Sectors has been used as an equity research tool by Indonesia's top financial institutions, and has also been used as a data source for academic research.
The depth of our data puts it head and shoulders above any comparable market intelligence tool, and is readily available through a simple developer experience with Sectors Financial API.

Beginners can start with the [Get your API Key](#get-your-api-key) guide, while more experienced developers can dive into our [API Reference](../../api-references/v2/indonesia/companies) for a comprehensive list of endpoints and examples.

We also have longer-form tutorial series and recipes for those looking to build financial analysis workflows and automations by example.

## Get your API Key

<Steps>
  <Step title="⏫ Upgrade to Sectors Insider">
    Through the Sectors Financial API, you are two lines of code away from Indonesia's most comprehensive financial market data and stock research tools. 
    Sectors Financial API is available to our [Insider plan](https://sectors.app/pricing) subscribers.

    <Frame>
      <img className="block" src="/images/pricing.png" alt="API Pricing" />
    </Frame>

  </Step>
  <Step title="🗝️ Obtain Sectors Financial API Key">
    Once [logged in](https://sectors.app/auth) with your Sectors account (Insider plan), navigate to the API Keys section in the [API page](https://sectors.app/api) to generate a new API key. 
    Include this API key in the header of your requests to authenticate and access the API (code snippets provided in the next step).

    <Frame>
      <img className="block" src="/images/key.png" alt="API Keys" />
    </Frame>

  </Step>
  <Step title="📊 Retrieve Data from the API">
    Use one of the documented API endpoints along with your Sectors Financial API Key to access data from Sectors. 
    Explore the [API References](../../api-references/v2/companies) to discover the available endpoints and sample responses.

    <CodeGroup>
    ```python Python
    import requests

    # Replace the URL with a URL from the API References
    url = "https://api.sectors.app/v2/.../"
    api_key = "API key acquired from the previous step"
    headers = {"Authorization": api_key}

    try:
        response = requests.get(url, headers=headers)
        response.raise_for_status()
        data = response.json()
    except requests.exceptions.HTTPError as err:
        raise SystemExit(err)
    ```

    ```js JavaScript
    // Replace the URL with a URL from the API References
    const url = "https://api.sectors.app/v2/.../"
    const api_key = "API key acquired from the previous step"
    const options = {method: 'GET', headers: {Authorization: api_key}};

    fetch(url, options)
      .then(response => response.json())
      .then(response => console.log(response))
      .catch(err => console.error(err));
    ```

    ```r R
    library(httr)
    library(jsonlite)

    # Replace the URL with a URL from the API References
    url <- "https://api.sectors.app/v2/.../"
    api_key <- "API key acquired from the previous step"
    headers <- c(Authorization = api_key)

    data <- GET(url, add_headers(headers))

    df <- fromJSON(content(data, "text", encoding="utf-8"),flatten=TRUE)
    ```
